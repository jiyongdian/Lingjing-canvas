## 🎨 设计元素

### 配色方案（来自应用主题）
```css
背景色: #F7F4EF (温暖奶油色)
主文字: #1F2421 (深色)
次要文字: #5C635D (灰色)
强调色: #C4612F (赤陶橙)
浅色面板: #FBF9F5
边框: #E7E1D7
```

### 字体建议
- **标题**: Fraunces / DM Serif Display (衬线字体)
- **正文**: Inter 300-500 (无衬线字体)
- **中文**: 思源宋体 / 思源黑体

### 必须包含的元素
1. ✅ 应用名称（中英文）
2. ✅ 版本号 (v1.3.2)
3. ✅ 平台支持 (macOS • Windows)
4. ✅ 核心功能关键词
5. ✅ 应用截图（至少一个）
6. ⭐ GitHub 仓库地址（可选）

### 图标建议
使用 Lucide 图标库中的图标：
- 🎨 节点/画布: `workflow` / `boxes`
- 📦 资源: `folder-open` / `image`
- 🤖 智能体: `bot` / `sparkles`
- 👥 协作: `users` / `share-2`
- 🔧 工具: `wrench` / `package`

## 📸 制作步骤

### 使用 Figma/Sketch/Photoshop

1. **创建画布**: 1280 x 640px，背景色 #F7F4EF
2. **添加截图**: 
   - 从 `docs/screenshots/` 中选择最佳截图
   - 添加轻微阴影增加层次感
3. **添加文字**:
   - 标题: 48-60px，衬线字体
   - 副标题: 24-32px
   - 描述文字: 16-20px
4. **添加图标和装饰元素**
5. **导出**: PNG 格式，优化压缩到 1MB 以下

### 使用截图 + 在线工具

推荐工具:
- Canva (https://www.canva.com)
- Figma (https://www.figma.com)
- Adobe Express (https://www.adobe.com/express)

模板尺寸选择: **Custom Size** → 1280 x 640px

## 🚀 快速制作方案

如果时间紧迫，可以使用以下简化方案：

1. **直接使用应用截图**
   - 选择 `docs/screenshots/01-canvas.png`
   - 裁剪到 2:1 比例
   - 添加简单的文字叠加层

2. **使用现有设计工具**
   - 在 Canva 搜索 "GitHub Social Preview" 模板
   - 替换为应用截图和品牌信息
   - 调整为建议的配色方案

## ✅ 检查清单

制作完成后，确保：
- [ ] 尺寸正确 (1280 x 640px)
- [ ] 文件大小 < 1MB
- [ ] 文字清晰可读（即使缩小显示）
- [ ] 品牌信息完整
- [ ] 视觉平衡，不过于拥挤
- [ ] 符合应用的设计风格
- [ ] 在深色和浅色背景下都能良好显示

## 📤 上传步骤

1. 将制作好的图片保存为 `social-preview.png`
2. 上传到仓库的 `docs/` 目录
3. 在 GitHub 仓库设置中：
   - Settings → General → Social preview
   - 点击 "Upload an image"
   - 选择 `docs/social-preview.png`
4. 预览效果，确认无误后保存

## 🎯 效果预期

完成后，当你在社交媒体（Twitter、Discord、微信等）分享仓库链接时，会自动显示这张预览图，让项目更专业、更吸引人！
