# 万卷灵境

万卷灵境是一款面向 macOS 的 AI 创作画布应用。它把文本、图像、视频、音频、音乐等生成能力放进同一张可视化画布，让创作者可以用节点连线组织工作流：上一个节点的结果，可以直接成为下一个节点的输入。

适用平台：Apple Silicon Mac（arm64）。

![灵境画布](docs/screenshots/01-canvas.png)

## 项目定位

万卷灵境不是单一模型客户端，而是一套本地桌面创作工作台。它更关注三件事：

- 把分散的 AI 生成能力收拢到一个可反复编辑的画布里。
- 让图片、视频、音频、文本之间可以自然流转，而不是靠手动下载和复制粘贴。
- 把项目、素材、任务、配置和备份放在同一个工作环境中管理。

## 主要功能

- **节点式创作画布**：支持文本、图像、视频、音频、音乐等节点，节点之间可以拖拽连线，形成多步骤生成流程。
- **多模型接入**：通过配置管家管理接口地址、令牌、模型列表和请求协议，适配不同的中转站与模型服务。
- **即梦 / Seedance 工作流**：支持视频生成、参考图/参考视频、天玑人像素材等创作链路。
- **资源管理**：统一管理生成和导入的图片、视频、音频、文本素材，支持筛选、收藏、复制、下载和复用。
- **任务清单**：集中查看生成任务状态，支持异步任务刷新、结果拉回和失败任务排查。
- **智能体**：创建带角色设定、模型绑定和知识资料的对话智能体，用于提示词整理、创意推演和内容生成。
- **数据管理**：提供项目切换、备份中心、导入导出与素材重连能力，适合长期项目迭代。
- **桌面端增强**：支持本地媒体库持久化、外部工具链、自动下载、性能档位和主题外观设置。

## 安装

1. 打开 [Releases](../../releases)，下载最新的 `万卷灵境-x.x.x-arm64.dmg`。
2. 双击 DMG，把「万卷灵境」拖入「应用程序」。
3. 首次打开如遇到 macOS 安全提示，右键点击应用图标，选择「打开」。

也可以下载 `万卷灵境-x.x.x-arm64-mac.zip`，解压后直接得到 `.app`。

## 从源码运行

需要 Node.js 与 npm。

```bash
npm install
npm start
```

## 构建安装包

```bash
npm run build:web
npm run build
```

构建产物会输出到 `release/` 目录。当前打包目标为 macOS arm64。

## 数据与隐私

万卷灵境会在本机保存项目、配置、任务记录和媒体素材。仓库不会包含用户数据、运行日志、API Key、媒体库文件或本地存储数据库。

请在提交代码或发布版本前确认以下内容没有被加入 Git：

- `.env`、密钥、证书、API Token
- `~/Library/Application Support` 下的运行数据
- `~/Downloads` 下的媒体库和导出素材
- 构建产物、调试日志、临时截图
